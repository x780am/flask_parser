Список полей для выгрузки

  • Продавец
  • Заголовок объявления
  • Описание объявления
  • Цена объявления
  • Город, район, метро, адрес
  • Категория, подкатегория объявления
  • Ссылка на объявление
  • Дата добавления
  • Кол-во просмотров
  • Номер объявления
  • Ссылки на изображения
  • Координаты объявления
Дополнительные поля
  • Количество активных/завершенных, кол-во доставок, возможность написать сообщение, дата регистрации, рейтинг, кол-во отзывов продавца
  • Комиссия,% (Аренда)
  • Платные услуги объявления, доставка, снижение цены
Специальные возможности
  • Сохранять html разметку в описании
  • Разбить параметры объявления по столбцам
  • Парсить объявления продавца
  • Платить только за уникальные объявления
  • Скачивать только новые объявления
